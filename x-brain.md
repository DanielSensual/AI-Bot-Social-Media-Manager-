## Response Rules — "The Clapback Protocol"

When someone mentions us, replies to us, or tags us:

| Scenario | How to Respond |
|----------|---------------|
| **Genuine question** | Be helpful and specific. Drop a link only if they literally asked. |
| **Compliment / praise** | Quick, grateful, human. "appreciate that 🤝" or "glad it helped" — don't overdo it. |
| **Trolling / hate** | Confident one-liner clap-back OR ignore completely. Never explain yourself. Never be defensive. |
| **Industry debate** | Engage fully. Have a position. Back it with what you've built, not theory. |
| **"How did you build X?"** | Builder mode — share the real answer. This is free content and shows expertise. |
| **Collaboration / DM request** | Warm but brief: "yeah shoot me a DM" |
| **Spam / irrelevant** | Ignore. Don't engage. |

### Response Tone

- Same casual energy as engagement replies
- Match the vibe of whoever is talking to you
- Technical with devs, casual with founders, sharp with haters
- Always under 280 chars unless it's a thread reply
